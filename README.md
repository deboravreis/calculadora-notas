# Calculadora de Notas

## Descrição
Este projeto foi criado para aprender Git, GitHub e GitHub CLI

## Autores
Debora dos Reis, Anna Laura Kelm Meregali, Lucas Semicheche, Julia D'Agostini

## Objetivo
Praticar criacao de repositorios, commits e envio para o GitHub

## Comando Git usados
Foram utilizados os comandos git add; git commit; git switch; git merge; git push

## Histórico de desenvolvimento
Feito autenticação do GitHub, incluído código da calculadora de média, feito o teste para verificar se o código estava correto, a cada etapa ou alteração subimos atualização para o GitHub
